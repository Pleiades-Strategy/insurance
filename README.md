# Insurance & Climate Risk — Legislative Tracker

A research tool built by [Pleiades Strategy](https://www.pleiadesstrategy.com) for tracking state-level insurance legislation related to climate risk. The project includes a public-facing landing page and interactive bill tracker, backed by a curated Airtable database covering 352 bills across 40 states.

**Live demo:** [pleiades-strategy.github.io/insurance_billtracking_demo](https://pleiades-strategy.github.io/insurance_billtracking_demo/)

## What it is

As climate-driven disasters reshape the insurance landscape, state legislatures are responding — sometimes to protect consumers, sometimes to protect insurers, sometimes to deny risk altogether. This tracker makes that activity visible and navigable.

The landing page (`index.html`) contextualizes the bill tracking work, highlights key findings from the 2025 legislative session, and links to forthcoming Pleiades research reports. The dashboard (`dashboard.html`) is an interactive browser for all tracked bills, with filters by state, status, and policy category, full bill text, sponsor details, and hearing records.

## Project structure

```
bill_collection/
├── index.html             # Public landing page
├── dashboard.html         # Interactive bill tracker (local, API-backed)
├── server.py              # Local API server (reads SQLite, serves JSON)
├── export_json.py         # Exports DB to docs/data.json for static hosting
├── docs/                  # GitHub Pages static site
│   ├── index.html         # Landing page (static)
│   ├── dashboard.html     # Bill tracker (static, reads data.json)
│   └── data.json          # Full data export (generated by export_json.py)
├── ingestion/
│   ├── airtable.py        # Airtable → SQLite sync (bills, sponsors, hearings)
│   └── legiscan.py        # LegiScan full bill text enrichment
├── cli/
│   └── ingest.py          # Main ingestion entry point
├── data/                  # SQLite database (gitignored)
└── config.py              # API keys and paths
```

## Setup (local development)

1. Clone the repo and create a virtual environment:
```bash
git clone https://github.com/Pleiades-Strategy/insurance_billtracking_demo.git
cd insurance_billtracking_demo
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and add your API keys:
```bash
cp .env.example .env
```

You'll need keys for [Airtable](https://airtable.com) and [LegiScan](https://legiscan.com/legiscan).

## Running the tracker locally

### 1. Sync data from Airtable

```bash
python cli/ingest.py --skip-text    # fast sync — metadata only
python cli/ingest.py                # full sync including LegiScan bill text
```

### 2. Start the local server

```bash
python server.py
```

This opens the landing page at `http://localhost:5050` automatically. The full interactive dashboard is at `http://localhost:5050/dashboard.html`.

## Updating the public demo (GitHub Pages)

After syncing new data, regenerate the static export and push:

```bash
python cli/ingest.py --skip-text
python export_json.py
git add docs/data.json
git commit -m "Refresh data"
git push
```

GitHub Pages redeploys automatically. The live site reflects the updated data within a minute or two.

## Data sources

- **Airtable**: Primary source of truth for bill metadata, sponsors, categories, and hearings — curated by the Pleiades research team
- **LegiScan**: Full legislative text fetched by state and bill number, decoded from base64 PDF/HTML
